# Privacy Policy — Chinese Hover Translator

_Last updated: 2026-07-15_

Chinese Hover Translator is a personal tool that shows pinyin, translation, and
context notes for Chinese text you hover on a web page.

## What it does with data

- **Hovered text:** When you hover Chinese text, the extension sends that snippet
  to a translation server **that you configure and control** (by default a server
  running on your own local network). It is not sent to the extension author or any
  third party.
- **Local storage:** Translations are cached and your server URL is saved in the
  browser's local extension storage, on your device, to avoid repeat requests. You
  can clear the cache anytime from the extension popup.

## What it does NOT do

- No analytics, tracking, advertising, or telemetry.
- No data is sent to the extension developer or any third-party service.
- No accounts, no personal information collected.

## Permissions

- **Host access / `<all_urls>`:** to detect Chinese text on the pages you visit.
- **Storage / unlimited storage:** to cache translations and save your settings locally.
- Network requests go only to the translation server URL you set.

## Contact

This is a personal/self-hosted extension. Questions: the operator of the
configured translation server.
